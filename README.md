# Szablony

Graficzny edytor szablonów oparty o sloty.
Aplikacja deskopowa w Pythonie (Tkinter + Pillow)

## Funkcje
-siatka szablonów
-obrazy i kolaże
-tekst automatycznie skalowany
-undo/redo
-łączenie projektów (+ i /)
-zapis do JPG + JSON

## Wymagania
-Python 3.9+
-Pillow

## Uruchomienie
bash
python main.py
